# 第166章 風狼的憂鬱 (完)

「畜牧之鄉」這個名字並非虛有其表。

在普蘭，牛和豬的數量遠遠超過人口，而位於深山中的托倫村，近年來喜事連連。

這個村莊本就老齡化嚴重，年輕人口流入幾乎為零。

活兒雖然多得干不完，卻沒人願意來這種窮鄉僻壤打工，村民們日復一日見到的都是熟面孔。

正因如此，任何一戶人家的喜事都會讓整個村莊熱鬧起來。

一戶人家的喜事就是全村的喜事。半天之內，消息就會傳遍全村，大家紛紛前來祝賀，這再正常不過了。

最近讓村莊沸騰的，是耶妮卡·佩洛弗的歸來。

雖然耶妮卡每次放假都會按時回家，但這次歸來卻格外熱鬧。

事情從耶妮卡·佩洛弗剛到家就開始了。

她回到佩洛弗牧場，換上村裡婦女的裝束，開始洗碗，像往常一樣幫忙牧場的工作，和父母一起享用羊奶、乳酪和黑麥麵包的晚餐，一切都顯得那麼自然。

然而，她的神情卻與往常有些不同。

佩洛弗牧場的主人奧爾泰·佩洛弗和他的妻子塞拉·佩洛弗比任何人都了解他們視如珍寶的獨生女耶妮卡·佩洛弗。

他們也知道，耶妮卡非常不擅長說謊。十幾年來，他們見證了耶妮卡的成長，僅憑她的表情就能看穿她的心思。

― 「最近看起來有些心事重重啊，耶妮卡。是不是因為從首席位置上退下來了？即使只是次席，也已經很了不起了。」

― 「啊，嗯？！才、才不是！雖然有點遺憾……但我並沒有因為這件事感到沮喪！爸爸真是的！」

坐在餐桌旁的耶妮卡極力否認奧爾泰的話，生怕父母擔心，不停地擺手。

這是她說實話時的反應。雖然她並沒有因為成績而感到壓力，但她的情緒卻明顯低落，這讓父母感到困惑。

老兩口多次注意到她望著虛空發獃，或是突然長嘆，彷彿在擔心什麼人。

― 「還是說……有喜歡的男孩子了？」

― 「啊？胡說什麼呢！爸爸怎麼總說些莫名其妙的話！什麼叫在意的男生，說得好像我有什麼非分之想似的！真是的！我、我去喂飼料了！雖然本來就要做...總之別突然說這種話！嚇死人了！」

耶妮卡的反應就像是在長篇大論地解釋「是的，沒錯」一樣，佩洛弗夫婦臉上露出了欣慰的笑容。

果然如此。耶妮卡·佩洛弗也到了這個年紀了。

佩洛弗夫婦在感情方面一直很開明。

這裡是連年輕人都難找的偏僻鄉村。他們一直擔心耶妮卡在這樣的地方長大，會變成一個不懂世事的鄉下姑娘。

牧場本就缺人手，要是能拐個女婿回來...夫婦倆的心裡充滿了期待。

佩洛弗夫婦對女婿的要求並不高。

雖然耶妮卡是個出類拔萃的女孩，但他們並不指望她能釣到什麼顯赫的貴族子弟、英俊的魔法師或人脈廣泛的名人。

他們只希望對方能熟練牧場的工作，手巧一些就好。只要耶妮卡不會在這偏僻的鄉村孤獨終老，或者因為年齡壓力隨便嫁個奇怪的人，他們就心滿意足了。

這樣的耶妮卡居然對西爾維尼亞的男生產生了好感！

女婿！女婿啊！

在這個日漸衰落的村莊里，他們從未敢奢望過的甜蜜辭彙。女婿！

就像發現了金銀財寶的海盜一樣，塞拉·佩洛弗眼中閃爍著光芒，緊緊抱住了耶妮卡。

平心而論，耶妮卡·佩洛弗確實是個無可挑剔的女孩。她長相甜美可愛，能力出眾，性格純樸善良，在這個時代簡直是稀世珍寶。

無論對方是誰，佩洛弗夫婦都已經做好了接受的準備。只要不是太過分的人，他們都會欣然接受。畢竟，西爾維尼亞的學生至少有一定的保障。

― 「耶妮卡！耶妮卡！你終於有出息了！」

― 「親愛的，別太激動了……不過，那孩子是什麼樣的？他會不會不喜歡太熱情的公婆？他是不是更喜歡有品位的感覺……？我們是不是該把結婚禮服重新改一下？」

― 「他會射箭嗎？我一直夢想著和女婿喝一杯，然後一起射箭……不過，現在喝酒還太早了。我希望他看起來有點男子氣概，而不是太瘦弱……你覺得呢？」

這興奮勁比喝十碗醒酒湯還上頭。看著父母誇張的反應，耶妮卡的表情逐漸凝固。

― 「不，不是的！為什麼你們會這麼想！」

― 「耶妮卡！別瞞著我們了，我們都知道了！所以，他叫什麼名字……？」

― 「才不說！憑什麼告訴你們！」

"才不說"而非"沒這人"的回答等於變相承認。畢竟這姑娘實在不會撒謊。

― 「你就告訴我們吧！耶妮卡！你知道我管理佩洛弗牧場有多辛苦嗎？我也想要點人生的樂趣啊！」

― 「爸爸……為什麼總是這麼激動……」

― 「牛糞要清理，羊奶要擠，這樣過一輩子也不是不行，但作為一個頂天立地的男人，我還是想要個女婿啊……！所以，至少告訴我們他的名字吧！男人最懂男人了！」

― 「夠了！夠了！拜託！小聲點！隔壁都聽到了！」

耶妮卡激動地拍著桌子，奧爾泰·佩洛弗嘆了口氣，垂下他那肌肉發達的手臂。

― 「好吧……既然你這麼說……那也沒辦法了……」

― 「親愛的，我們不是早就做好心理準備了嗎？耶妮卡不可能永遠保持她那純真善良的性格……她遲早會進入青春期的，我們不是在她出生那天就下定決心了嗎？別太傷心了，這只是命中注定的時刻到了而已。」

― 「是啊……你說得對……每個人都會經歷這個過程。現在耶妮卡也會因為我的衣服和她的衣服混在一起洗而生氣，或者一看到我就皺眉頭，或者砰地一聲關上門躲進房間，或者敲著桌子要零花錢……雖然我早就做好了心理準備，但……這真是件讓人傷心的事啊……我也要變成角落裡發臭的老頭子了……是啊，這就是人生的黃昏期吧……真是苦澀又孤獨啊……」

― 「你們為什麼總是把事情想得那麼複雜……！我不會那樣的，不會的……！」

看到奧爾泰沮喪的表情，耶妮卡一時語塞，猶豫了起來。無論如何，名字是絕對不能說的。

在這個狹小的鄉村社會裡，一旦名字被說出口，消息瞬間就會傳遍全村。作為在托倫村生活了多年的耶妮卡，她比誰都清楚這一點。

然而，父母那沮喪的表情不斷施加壓力……她的額頭上開始滲出冷汗。

「呃……那個……」

最終，善良的耶妮卡支支吾吾地開口了。

「他……射箭很厲害……」

看到奧爾泰臉上露出喜色，耶妮卡感覺血液直衝腦門。

― 「哦，耶妮卡！我聽說了！你在學校過得很好，我真為你高興！」

― 「什麼時候帶他回來一趟！我有珍藏了二十多年的覆盆子酒！」

― 「那我得宰一頭豬！」

― 「一頭豬怎麼夠！得宰一頭牛！我會在牧場上挑一頭最好的，等他來了就宰了！」

雖然她沒有說出名字，但確實說漏了嘴。

佩洛弗夫婦這種糊塗蟲不可能不把這種大事告訴村裡的人。

奧爾泰在村公所喝了幾杯，心情大好，把和耶妮卡的對話說了出去，消息瞬間傳遍了全村。

第二天傍晚，整個村莊都沉浸在喜事中，耶妮卡提著裝滿水果的籃子走在村廣場上，臉上火辣辣的。

― 「哦，等他來了，一定要來我們赫爾肯雜貨店！雖然沒什麼好招待的……但如果有需要修補的東西，我們可以免費幫忙！」

― 「西爾維尼亞的學生，應該是有錢人家或者貴族吧？雖然不是所有人都這樣，但這樣的人不少！耶妮卡，你真是出息了！你是我們村的驕傲！」

― 「耶妮卡！耶妮卡！耶妮卡！耶妮卡！耶妮卡！」

― 「哎呀，真是時候！我二十五年前也經常在城裡勾搭男人呢……真是懷念啊~」

被祝賀了一整天的耶妮卡，第二天早餐時對佩洛弗夫婦宣布：

― 「我明天就回西爾維尼亞。」

這消息如同晴天霹靂。耶妮卡每次回家至少會待上半個月，但這次她只待了三天就宣布要回學校。

― 「發，發生什麼事了，耶妮卡。再多待幾天吧。」

― 「我……」

耶妮卡的臉紅得像熟透的蘋果，含著眼淚說道：

― 「我在這裡待不下去了……」

對於這對鄉村牧場的糊塗夫婦來說，少女細膩的心思實在難以理解。

...真是聞者傷心。

[ 突然想起些往事 ]

北邊森林的中央有一個相當大的湖泊。

這湖泊非常神奇。雖然湖水的規模並不算大，但湖水卻從未渾濁或腐壞。

死水通常會逐漸變得渾濁，但這湖水卻像是有人在不斷凈化一樣……中央湖泊的水總是清澈而神秘。

尤其清晨時分，氤氳霧氣中若隱若現的湖面被晨光鍍上金邊，宛如童話場景，美不勝收。

有些學生甚至把這裡當作晨練的路線。尤其是像直斯這樣的傢伙，經常光顧這裡。

不過那是早晨……現在是夜晚，夜空的星光倒映在湖面上，閃爍著微弱的光芒。

中央小島般的草坪上矗立著「梅麗達的守護樹」。

倚靠樹榦眺望森林夜景時，彷彿天地間有兩個夜空。明鏡止水映出的星空，正以假亂真地鋪陳著璀璨星光。

松樹林的某處傳來夏夜特有的蟲鳴聲，或是小動物在草叢中穿梭的聲音。

夏夜從來不會寂靜。與彷彿被世界遺棄的冬夜截然不同。

這片看似只有樹木的森林裡，確實充滿了生命的氣息。

「想起了過去？」

[ 只是偶爾會這樣。 ]

梅麗達在湖面上輕輕走著。由於她的現形已經解除，所以才能這麼做。若非法杖的力量，魔力枯竭的我根本看不見她。

看著她赤腳在倒映著星光的湖面上行走，彷彿她不是在湖面上，而是在天空中漫步。

[ 不久前，我們不是剛剛摧毀了羅斯泰勒家族嗎？ ]

「說摧毀有點過分了。」

[ 有什麼關係？這是事實。 ]

梅麗達輕輕提起裙擺，轉了個圈，微風輕輕拂過。

[ 你可能不這麼想，但對我來說，羅斯泰勒家族是一個歷史悠久、聲名顯赫的家族。 ]

「我並沒有不同的看法。」

[ 也許吧。但知道它的歷史和親眼見證它的衰落是兩回事。 ]

「這倒是沒錯。」

我爽快地承認了這一點，梅麗達笑著點了點頭，彷彿早就料到了我的反應。

[ 每次看到這種龐然大物消失，就會真切感受到時代變遷。該怎麼說呢...特別令人泄氣。 ]

「嗯……」

[ 你不理解嗎？ ]

「老實說，確實不太理解。」

梅麗達笑著繼續說道。

[ 有時候，我覺得時代把我拋在了後面。也許是因為作為精靈，我的生命已經進入了黃昏期。 ]

黃昏期。

聽到這個詞，我一時語塞，需要整理一下思緒。

梅麗達是一個活了很久的高位精靈。

雖然人們普遍認為精靈沒有壽命的概念，但這是誤解。

精靈不會因為物理衝擊或魔法力量而死亡，但任何精靈都無法在現世度過超過其魔力感應允許的時間。

延長這個時間的唯一方法就是提升位階。

從只能維持一周的流體精靈，到可以維持幾年的低階精靈。

再到可以輕鬆維持幾十年的中階精靈，甚至是可以活上幾百年而不消散的高位精靈。

然而，無論多麼強大的精靈，最終都必須回歸自然的循環。

雖然沒有死亡的概念，但回歸自然的概念依然存在。雖然這與死亡沒有太大區別，但至少可以抱有希望，未來或許能再次以流體精靈的身份重新開始。

精靈的生命是如此漫長而單調。

「梅麗達，你……」

[ 別誤會，我還有很長時間。至少還有一百多年。 ]

她說著，露出了狼一般的微笑。

<hr/>

[ 怎麼？擔心我馬上就會離開嗎？呵呵。 ]

「...否認就是撒謊了」

[ 哇哦，在這種微妙的地方倒是很誠實嘛。你其實是個很重感情的人呢。 ]

「不管怎麼說，我確實從你那裡得到了很多幫助。我還不至於那麼冷血。」

我靠在梅麗達的守護樹上，茫然地望著天空。

「我只是好奇，為什麼你突然變得這麼憂鬱。現在我明白了。」

[ 憂鬱？我？ ]

「別裝了。你現在就是很憂鬱。難道還要我明說嗎？」

無論是精靈還是人類，擁有心靈的本質是相同的。而且我個人認為，心靈運作的方式在人類、動物甚至精靈之間並沒有太大區別。

梅麗達看起來情緒低落、心事重重，原因也差不多。

「人類也是一樣的。」

我嘆了口氣，繼續說道。

無論是在戰場上，還是在和平的社會中，這都是普遍適用的道理。

年輕氣盛的低階精靈們總是眺望"未來"。

他們夢想著成為中階精靈，提升魔力感應，增加魔力，最終成為高位精靈……指揮其他精靈，與強大的精靈使一起週遊世界。

然而，活得足夠久……即將進入黃昏期的精靈們，則會回顧「過去」。

他們回憶曾經遇到的精靈使，見過的人類，經歷過的時代。

衡量過去的日子和未來的日子，就能輕易地知道自己是什麼樣的存在。

未來日子更多的人會展望未來，而過去日子更多的人則會回憶過去。

剛畢業的年輕人和即將退休的老者無法溝通，原因也是如此。

年輕人談論未來，而老紳士只談論過去的經歷。意識到這種錯位後，老者會逐漸陷入憂鬱。

-我的故事快要結束了。時代終究把我拋在了後面。

每當意識到這一點，即使只是看著夕陽，也會讓人淚流滿面。

[ 她很像西爾維尼亞，那個女孩。 ]

突然，話題轉向了大賢者西爾維尼亞。

雖然梅麗達的外貌與西爾維尼亞如出一轍，但我知道她說的並不是外貌，於是閉上了嘴。

佩特里西亞娜是個連學院消息都不知道，只埋頭於自己研究的怪人。

[ 那位大賢者也是個在惡劣環境中依然執著於魔法研究的怪人。 ]

「所以你才和她走得近？」

[ 我只是想觀察她。因為想起了過去。雖然在這個過程中給你添了些麻煩。 ]

「這點麻煩不算什麼。特蕾西亞娜也來幫忙解決了。」

梅麗達笑著歪了歪頭。

[ 結果來看，她和當年的西爾維尼亞還是有很多不同的地方。當然，這也是理所當然的。那樣的怪人可不多見。 ]

「是嗎。」

[ 嗯，是的。就像你說的，我可能確實有點憂鬱。 ]

梅麗達展開裙擺，站在湖面上，眺望著西爾維尼亞的尖塔。夜空的星光下，高聳的尖塔不止一兩座。

從北邊森林的湖泊也能清晰看到的西爾維尼亞尖塔，是當年被流放到阿肯島的一位怪人魔法師留下的遺產。

雖然起點微不足道，但如今它已成為大陸上最頂尖的教育機構。

那漫長的歲月雖然看似遙遠，但回憶起來，卻彷彿就在昨天。

梅麗達積累的那些記憶，如今在進入黃昏期的她心中依然清晰。

正如所說，進入黃昏期的人更傾向於回顧過去，而不是展望未來。

梅麗達閉上眼睛，回憶起的那些記憶也是如此。在她漫長的生命中，那些始終無法忘記的記憶，對她來說格外珍貴。

在山頂俯瞰阿肯島的日子。

因為西爾維尼亞的惡作劇而頭疼的日子。

西爾維尼亞雙手叉腰，豪爽大笑的樣子……確實讓人難以忘懷。

[ 回想起來，確實……再也沒有像她那樣的朋友了。 ]

梅麗達低聲呢喃著，臉上露出了與憂鬱無關的平靜表情。

[ 嗯，那時候真的很美好。 ]

雖然不能僅憑表情來判斷，但她的確如此想的。

*

[ 有一個最高階風精靈的遺骸。 ]

即使不打開狀態欄，我也能感覺到與梅麗達的親密度提升了。

與其說是狀態欄上的親密度，不如說是精靈與精靈使之間的共鳴感更加強烈了。畢竟我們一起經歷了不少冒險，也經常交流。

雖然親密度提升後，精靈術和技能的威力也會增強，但梅麗達在與我進行更深入的對話後，透露了一個更高層次的信息。

[ 現在最高階風精靈的位置是空缺的。雖然幾十年過去了，但沒有人能接近那個境界。只有一位壽命耗盡、回歸自然的前最高位精靈留下的遺骸。 ]

梅麗達坐在她的守護樹旁，抱著膝蓋說道。

[ 其實說是遺骸也不太準確。它更像是一團小小的魔力絲線。精靈本來就沒有遺骸這種東西，但它消失時留下的痕迹，勉強可以算是遺骸吧。 ]

梅麗達展開手掌，對著夜空繼續說道。

[ 如果得到它，你的精靈術修鍊會有巨大的進步。西爾維尼亞研究完後把它埋在了地下，如果你想要，我可以把它交給你。 ]

「你怎麼突然有這種想法了？」

[ 只是覺得，像你這樣的朋友很難得。耶妮卡也是個很好的朋友，但她的內心並沒有那麼深沉，無法像你一樣讓我敞開心扉。 ]

最高階風精靈「提爾卡拉克斯」——山嶽般龐大的巨熊形態精靈，如今只存在於歷史典籍中。

[ 不過，這不是免費的。 ]

<hr/>

梅麗達露出了狡黠的笑容。

「你又在打什麼奇怪的主意了。」

[ 完全沒有~。就像我說的，我有點憂鬱。所以我想讓你幫我緩解一下這種憂鬱感。 ]

「你還想要什麼？」

梅麗達突然直起身子，拉住我的領帶，在我耳邊低聲說道。

[ 你知道耶妮卡喜歡你吧？男女那種。 ]

這突如其來的話讓我一時語塞。

[ 雖然理解你一直很忙...但現在總算安定下來，也有空閑了吧？ ]

「你到底想說什麼？」

[ 該推進一下關係了。 ]

她一邊說著，一邊用手指繞著領帶，用嫵媚的聲音說道。

[ 在假期結束之前，你得親耶妮卡一下。 ]

「……」

我一臉無語地盯著梅麗達，她甩了甩她那雪白的頭髮，露出了狡黠的笑容。

[ 那個商隊的狐狸精要是住進你的營地，耶妮卡肯定會輸的。得在事情發生之前，先把關係定下來，這樣我才能安心。 ]

「……」

[ 成交？ ]

暑假已經接近尾聲。

再過不久，學生們就會陸續返回學院。

不知為何，一種不祥的預感湧上心頭，我盯著梅麗達那狡黠的臉，久久沒有回應。
