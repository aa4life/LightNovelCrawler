# 第50章 賢者之書爭奪戰（5）

[ 他看起來真的很辛苦。 ]

風吹過。森林中自然的風總是讓人心情愉悅，但拂過耶妮卡臉頰的這陣風並非自然之風。

高階風精靈梅麗達。

她那棕色的毛髮隨風飄動，看起來輕盈無比，但她的體型卻絕不輕巧。

梅麗達靠在守護樹上，低聲對耶妮卡說道。

[ 耶妮卡，你也知道學院的課程並不輕鬆吧？即使是年級第一的你也忙得不可開交，更何況他還得兼顧營地生活，自然會被逼到極限。 ]

「梅麗達也這麼想啊。」

與對待塔坎時的態度截然不同。

耶妮卡抱著膝蓋，認真聽著梅麗達的話，咽了咽口水，專註地聽著摯友的建議。

[ 不過，這是個機會，耶妮卡。那隻狐狸現在忙著談判什麼的，沒空來營地。你得抓住這個機會。 ]

「嗯，嗯。」

[ 人在極限時更容易被攻陷。別覺得良心不安，好好利用這次危機。你得讓他徹底淪陷，所以可以表現得狡猾一點，甚至勢利一點。還需要一些表演。 ]

「能具體說說嗎？」

[ 嗯…… ]

梅麗達雖然外表粗獷，但言行舉止卻相當細膩和端莊。不過，她的思維方式卻並非如此。

[ 無論你喜不喜歡，想要抓住男人的心，就得學會像狐狸一樣狡猾。在這方面，那隻狐狸般的少女確實聰明。你得學學她的優點。 ]

梅麗達盤腿坐著，抬起頭，清了清嗓子，發出「嗯、哼、呃」的聲音。

[ 比如說，穿著。 ]

梅麗達的戀愛建議如同黃金般珍貴。耶妮卡這麼想著，不禁有些驚訝。

她原本打算在篝火旁嘗試做些料理，所以從學院食堂拿了一些快要過期的食材。本來這是不允許的，但貝爾理解她的處境，睜一隻眼閉一隻眼。

因為要做飯，她的穿著非常簡單。她捲起那件洗過多次的襯衫袖子，繫緊從家鄉牧場帶來的深藍色裙子，披著披肩。

她想起了之前見過的洛特爾那華麗而美麗的褶邊連衣裙。平時總是扎在一側的紅褐色頭髮散開，戴著藍色玫瑰髮飾和琥珀石首飾的模樣……即使是同為女性的她看了也不禁讚歎。

再看看自己現在的樣子……耶妮卡的形象更像是個充滿生活氣息的村婦，而不是個有魅力的女性。與她平時穿的整潔可愛的校服形成了鮮明的對比。

[ 滿分，耶妮卡。你也快成狐狸了。 ]

「嗯？」

然而，自稱戀愛天才（實際戀愛經驗為零）的梅麗達卻給予了高度評價。

「我現在看起來不是很邋遢嗎？」

[ 耶妮卡，你也挺會裝傻的。你有你自己的魅力。和平時活潑開朗的樣子不同，這種充滿生活氣息的反差……男人很容易被這種反差吸引。裝作若無其事地展現這種樣子，真的很棒。 ]

「真，真的嗎……？」

[ 最好再把袖子卷高點，頭髮也紮起來。給人一種能共患難的賢內助感覺，真的很不錯。 ]

「嗯，嗯！對！這種時候就得裝得若無其事！」

耶妮卡像是從一開始就計畫好似的，把袖子卷得更高。梅麗達在心裡嘆了口氣。耶妮卡似乎覺得自己完美地演出了「若無其事」的樣子。

如果她能像洛特爾那樣表現出一點市儈的樣子就好了，但不出所料，這姑娘現在才開始調整表情微笑。她就像一張白紙。

這樣下去怎麼談戀愛？

再這樣下去，她只會淪為三流戀愛小說里被利用後拋棄的悲情女配角。

梅麗達對人類的文化產物很感興趣。

她有著能判斷出連普通人類都難以理解的藝術品價值的眼光，偶爾還會用不熟練的變形魔法模仿人類形態，去圖書館借書。

梅麗達想起了偶爾翻到的三流戀愛小說。耶妮卡簡直就像是那些小說里的悲情女主角的翻版。

總是呼喚著過去戀人的名字，咬著手帕哭泣的經典橋段不斷浮現在腦海中，讓她不禁感到一陣無奈。

她並不指望耶妮卡能像那個商人一樣優雅而高貴地調情。

但要想吸引男人，至少得讓他渴望得到你吧？

[ 總之你今天要去看他吧？如果浪費了這麼好的機會，就算老天原諒你，我也絕對不會原諒你。所以，今天去了一定要有所進展，明白嗎？ ]

「進，進展是指……」

[ 不，現在就跟我說。耶妮卡。你今天去營地幫他，打算做點什麼？你打算怎麼拉近和他的距離？ ]

「呃……那個……」

耶妮卡避開視線，尷尬地說道。

「啊，沒什麼。我本來沒打算做什麼。」

[ 啊啊啊啊啊！ ]

梅麗達強忍著內心的怒火，繼續追問耶妮卡。

[ 這說得通嗎！這種時候，這種機會！你覺得那個木頭人還會再有這麼辛苦的時候嗎？看看他的成長速度！簡直快得離譜！這種機會怎麼可能輕易再來！共患難的立場是最卑鄙的必勝公式！你得利用這個位置，哪怕顯得市儈一點！哪怕顯得狡猾一點！ ]

「那，那是當然。所以……我本來沒打算做什麼。」

[ 這到底是什麼鬼話！ ]

「不是，我是說……只是……這是我的想法……」

聽完耶妮卡接下來的解釋，梅麗達氣得說不出話來。

然而，冷靜下來想了想……這確實很符合耶妮卡的風格。

甚至，這種特質可能會成為她的優勢。想到這裡，梅麗達點了點頭。

[ 是啊，真的很像你。 ]

聽到這話，耶妮卡害羞地低下了頭。

「黃金之女這個稱號很快就要成為過去式了。等埃爾特正式失勢，這個名號也就不復存在了。」

克雷平·羅斯泰勒親自來到埃爾特商會的接待室，這是洛特爾始料未及的。

她只能儘快完成接待準備，坐在座位上迎接他。

「您親自來到我這個貪婪商人的寒舍，真是讓我既感到榮幸又擔心。不知道我們的店員有沒有失禮之處。」

<hr/>

「沒什麼特別的。」

克雷平坐在接待桌對面，將茶杯端到嘴邊。

雖然商會準備了最高品質的茶，但對克雷平來說，這不過是普通的商品。

洛爾特坐在克雷平對面，微微低下頭。

她的每一口呼吸都經過精心計算，甚至連吞咽口水都小心翼翼，生怕露出破綻。

克雷平·羅斯泰勒是個不容小覷的對手。

不過，既然已經大致猜到了他的意圖，也就沒必要過度緊張。

「我會從埃爾特商會手中買下賢者之書。」

他的語氣彷彿理所當然地認為埃爾特商會會出售這本書。

「告訴我你預期的利潤，我會在此基礎上加價。你一開始就是這麼打算的吧？」

埃爾特商會之所以在競標時抬高價格，是因為他們確信克雷平·羅斯泰勒會回購。

這一點……似乎連克雷平都知道了。

「不過……有點奇怪。」

克雷平低下頭，低聲繼續說道。

「你怎麼會提前知道我會如此迫切地想要得到賢者之書？據我所知，埃爾特商會從不做這種高風險的投資。」

「誰知道呢。」

「感覺像是有人把我的意圖透露給了你。」

洛特爾沒有犯下愚蠢的錯誤，比如吞口水或結巴。

不過，克雷平的話確實切中了要害。

「你們有了解我們家族內部情況的消息來源嗎？」

「是的，可以這麼說。」

拙劣的謊言只會讓對方產生新的懷疑。

如果在這裡草率行事，埃德·羅斯泰勒可能會受到牽連。

商人利用消息來源謀利並不奇怪。以克雷平的性格，他不會因此與埃爾特商會敵對。不過，對於內部的背叛者，他絕不會輕易放過。

即使是以仁慈著稱的克雷平公爵，也不會對已經被逐出家族卻仍在出賣家族秘密的埃德無動於衷。

「不過，消息來源離得太遠，溝通起來很麻煩，而且對方身份高貴，要求的報酬也很高。所以，我希望至少能收回投資成本……這就是商人的心思。」

謊言中摻雜一點真相時，欺騙效果會更好。

與其一味地否認，不如巧妙地引導對方走向與真相不同的方向。

判斷對方的話是真是假很容易。但要明確區分真相與謊言的界限……即使是直覺敏銳的人也難以做到。

洛特爾是個善於利用這一點的談判高手。

「你打算報多少？」

「12000普倫金幣。」

「好。等學院移交賢者之書後，我會立即買下。」

沒有冗長的附加談判。表面上，這是一筆非常乾脆的交易。

畢竟，正式從學院獲得賢者之書的是埃爾特商會，只有完成移交後，埃爾特商會才有出售權。在此之前，只能等待。

「啊，還有件事我得提前告訴你。免得事情變得更複雜，讓我頭疼。」

「哦，是什麼？」

「關於你養父埃爾特·凱赫倫的事。雖然他已經失勢，不再是我的關注對象，但如果他的事影響到我的計畫，我會很麻煩。」

克雷平放下茶杯，低聲繼續說道。

「據說他在本家失勢後，集結了所剩無幾的個人資產，僱傭了一支傭兵團。我昨天收到報告，他們正經過我的領地，前往西爾維尼亞。」

「什麼？」

洛特爾也意識到埃爾特的動向有些可疑，但沒想到他已經採取了行動。

「不過，他應該不會在受帝國保護的西爾維尼亞學院鬧事……但他顯然有所打算。希望你不要被這些外部因素拖累，耽誤了正事。」

說完，克雷平帶著隨從離開了埃爾特商會的接待室。

洛特爾送走克雷平後，獨自坐在接待室里，抿了一口剩下的茶。她需要整理一下思緒。

無論如何，賢者之書的轉售日程已經確定。

埃德最初的請求只是讓她買下「賢者之書」的實物……她並沒有違背他的請求。

埃德也知道她計畫轉售賢者之書。因此，她算是完成了他的請求。

所以，只要完成書的購買，她就可以向埃德提出「任何」要求。

這是多麼甜蜜的誘惑啊。距離那一刻已經不遠了。

「任何」這個簡單的詞讓洛爾特的少女想像力開始飛揚。

如果真的可以……這樣也行……那樣也行……

就在她沉浸在美好的幻想中時，思緒突然轉向了另一個方向，洛特爾的冷靜瞬間回歸。

她得到了一個不容忽視的信息。

埃爾特·凱赫倫要來了。

說實話，她並不害怕。這不過是他在墜入深淵前的最後掙扎。

他以為僱傭一支傭兵團就能對洛特爾使用武力，那就大錯特錯了。

首先，西爾維尼亞學院內不可能允許大規模的私人武裝進入，而洛特爾個人的武力也絕不弱。

「如果他有別的打算……難道是打算抓人質？」

想到這裡，她有了頭緒。

「你以為因為那孩子對你有好感，我就會特別優待你嗎？還是說，你被那孩子漂亮的外表迷惑了？」

埃爾特似乎誤以為埃德對洛特爾有好感。

<hr/>

雖然事實恰恰相反，但在極端危機時刻，他們依然互相支持。足以讓人覺得他們關係不斐。

儘管洛特爾身邊總是有商會的護衛，她個人的武力也不弱……但如果目標是埃德，情況就不同了。

如果埃爾特打算綁架埃德作為談判籌碼……

「他越界了。」

在昏暗的接待室角落裡，洛特爾披上了長袍。

「鏘！看起來很好吃吧？」

我躺了一整天。

雖然想干點什麼，但總是頭暈目眩，感覺還得再休息幾天。

最讓我感激的是耶妮卡的幫助。

進入恢複狀態的第二天。耶妮卡從某處弄來了一大堆食材，用相當熟練的手藝做了很多料理。

夜已深。因為是周末前夜，耶妮卡似乎沒有回宿舍的意思。

篝火的光芒艱難地驅散了四周的黑暗。

夜晚總是歡快鳴叫的蟋蟀聲隨著秋天的到來逐漸減少，取而代之的是貓頭鷹的叫聲和樹葉在風中沙沙作響的聲音。

耶妮卡做的燉菜確實比直斯做的要美味得多。

雖然覺得有必要學點烹飪技巧，但現在還是以恢複為主，訓練的事以後再說。

「小時候在牧場，我每天都做燉菜。你知道嗎，埃德？我小時候養牛的時候總是……」

耶妮卡笑嘻嘻地坐在篝火旁，滔滔不絕地講著故事。我靜靜地閉上眼睛。雖然身體正在恢複，但體力還沒有完全恢複。

「你看起來很累，埃德。」

「是啊。」

「該休息了。嗯。」

「你幫了我這麼多，我卻只讓你看到我疲憊的樣子，真是抱歉，耶妮卡。」

「嗯？完全沒有。我沒做什麼特別的事。」

做飯和打掃房間對耶妮卡來說似乎不算什麼。

她照顧病人的能力看起來很熟練，問起來才知道，她在牧場工作的父親經常受傷，所以她經常幫忙。

感覺又有點發燒，摸了摸額頭，確實有些燙。頭痛和低燒似乎又來了，我不由得嘆了口氣。

我走進小屋，躺在簡易床上，耶妮卡也跟著進來，坐在角落裡，笑嘻嘻地看著我。

「……你想幹嘛？」

「什麼都不幹啊？」

「什麼都不幹？」

「我只是想陪著你。」

我無奈地看著她，她卻一臉理所當然。

「埃德，我知道你現在很辛苦。身體累了，心也會累，誰都一樣。我知道你很努力，比誰都辛苦。所以，你不用假裝不累。」

她挺起胸膛，像是在等我誇她。

「這和你坐在這兒什麼都不幹有什麼關係？」

「你不知道什麼都不幹、只是陪著有多大的力量嗎？埃德，你沒經歷過所以不懂！」

她低下頭，低聲說道。

「我也有過很辛苦的時候。雖然得到了很多安慰和幫助……但最後給我最大力量的，不是那些急著安慰我或幫我做事的人……而是那些默默陪在我身邊的人。我覺得是這樣，所以對你一定也有用！你先試試！」

說完，耶妮卡走到我躺著的床邊……然後直接靠在床架上，靜靜地坐了下來。我擔心地板會冷，但看來還沒到冬天，還不至於那麼冷。

我躺在床上，耶妮卡坐在地上，雖然看起來不太合適，但她毫不在意。

「你辛苦了，做得很好。」

她只是閉著眼睛，微笑著。

「不，你不用這樣……」

「噓！」

她生氣的樣子與其說是兇悍，不如說是可愛，讓人忍不住想笑。

「你很辛苦！承認了才能好好休息！承認吧！埃德，你很辛苦！」

「……」

我靠在床背上，慢慢地……低聲說道。

也許連我自己都不願承認這個事實。

「是啊……我很累。」

一種奇怪的感覺湧上心頭。

過去的半個學期，我幾乎是以小時為單位生活的。有沒有哪一天我沒有感到疲憊？我一時想不起來。

「真的很累。太累了。」

我感覺終於可以好好睡一覺了，慢慢地放鬆了身體。

就這樣，我睡了整整十個小時，醒來時身體終於恢複了。

心理上也似乎完全恢複了，感覺神清氣爽。

剛醒來，我就看到耶妮卡靠在我的膝蓋旁，呼呼大睡。

看到她流著口水、睡得昏昏沉沉的樣子，我忍不住笑了出來。

這真是一次意外的「襲擊」。

-『賢者之書出售決定。特里克斯特館負責人需完成感應者交接準備並提交報告。』

負責學院魔法書感應者交接工作的是格拉斯特教授。

實際上，大部分工作都交給了圖書管理員，他只是掛了個空頭銜。

<hr/>

不過，這種規模的交接工作，必須由最高負責人親自處理。

「格拉斯特教授！負責賢者之書的工作人員在找您！」

助理教授克萊爾猛地推開格拉斯特教授辦公室的門，走了進來。

克萊爾看到的景象是，格拉斯特教授靜靜地坐在教授椅上，望著窗外的大片風景。

格拉斯特教授的辦公桌上已經堆滿了與賢者之書出售相關的報告文件。

「啊，您已經收到報告了？那文件處理完就向學院彙報……」

克萊爾突然意識到，格拉斯特教授沒有任何反應，於是又喊了一聲他的名字。

格拉斯特教授依舊靜靜地坐在那裡，望著窗外。
