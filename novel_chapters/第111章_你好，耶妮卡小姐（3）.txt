# 第111章 你好，耶妮卡小姐（3）

從皇都出發，騎馬日夜兼程三天才能到達的東部斯帕爾德地區，是一片畜牧業發達的地方。

克羅艾爾帝國流通的畜產品中，有四成產自斯帕爾德地區，而著名的「畜牧之地普蘭」正是位於斯帕爾德地區南部的山脈地帶。

深入普蘭的山谷，沿著山脊附近望去，可以看到一個不到三百人的小村莊——托倫。

雖然村莊規模很小，但這裡生產的畜產品數量卻遠超村莊的規模，足以養活一個小城市。村莊里一半的人口從事畜牧業，另一半則從事畜產品的加工業……這是一個位於山谷中的小型畜牧村莊。

在村莊外圍，有一個規模適中的牧場——已經傳承了四代的佩洛弗牧場。這個牧場的獨生女，正是托倫村的驕傲。

從咿呀學語時就能看到精靈的少女耶妮卡·佩洛弗，是村裡公認的天才中的天才。

這個位於山谷中的畜牧村莊，居民們逐漸老齡化，嚮往城市的年輕人也漸漸失去了聯繫。

用好的話說，這是一個和平而寧靜的村莊；用不好的話說，這是一個單調乏味的村莊。

在這樣的封閉世界裡，擁有非凡能力的耶妮卡度過了怎樣的童年，幾乎可以想像。她必然獨享了全村人的寵愛。在充滿愛的環境中，她自然成長為一個性格溫和、沒有稜角的女孩。

她常常抱著裝滿雞蛋的籃子，和小小的精靈一起在村莊廣場上漫步，將新鮮的食品分發給鄰居們。

她活潑地在圍牆間奔跑，村民們熱情地向她打招呼……這些記憶在耶妮卡的腦海中依然清晰。

天空中飄著幾朵懶散的雲，覆蓋著翠綠草木的山脈風景一如既往。

村莊里進出的外人，大多是偶爾來討價還價的行商、物流工人或郵遞員。

在這個村莊里生活了十幾年，耶妮卡只能看到那些熟悉的面孔。

隔壁的杜林先生，對面的雷特先生，廣場旁的阿倫先生，村長阿爾克斯先生。

耶妮卡在村莊里度過的童年時光里，周圍幾乎都是長輩。

這個逐漸老去的山谷村莊，出生率低得可憐。耶妮卡幾乎一直是村裡最受寵的小女兒。

因此，她從未體驗過被人恭敬地尊稱的感覺。

離開家鄉，進入西爾維尼亞學院已經三年了。雖然已經逐漸適應了這裡的生活，但對還未完全長大的耶妮卡來說，仍然有許多陌生的事物。其中之一就是那種恭敬的語氣。

光是適應奧菲利斯館女僕們的恭敬對待，就花了將近半年的時間。

「你好，耶妮卡小姐。」

突然，耶妮卡被叫到名字，立刻回過神來。這裡是特里克斯特館的行政諮詢室。

「今天又來了呢。已經不知道是第幾天了……如果有任何諮詢或申請，請直接告訴我。如果是我們行政處理範圍內的事情，我們會儘快處理。」

「啊，那個……我……」

在彬彬有禮的行政職員面前，耶妮卡一時語塞。她猶豫著是否要提交藏在袖子里的文件，最終還是搖了搖頭。

行政職員嘆了口氣，點了點頭。

「雖然不知道您在煩惱什麼……但如果實在不知道該怎麼辦，不妨和周圍的人商量一下，耶妮卡小姐。如果您說您有困難，需要諮詢，學院里會有很多人願意幫忙的。」

「啊，哈哈……和別人商量這種事有點難……」

耶妮卡含糊其辭地應付了幾句，然後像逃跑一樣離開了行政諮詢室。

她拿出原本想提交的文件，嘆了口氣。

「總是忙著看好的東西，想著好的事情……為什麼不好的想法總是在有空隙的時候鑽進來呢？」

耶妮卡搖了搖頭，重新整理了自己的思緒。

最近，她的學院生活過得非常舒適和幸福。

課程勉強能應付，和朋友們相處融洽，還有許多仰慕她的人，宿舍設施也不錯，甚至還能和暗戀的男孩整天待在一起。雖然今天早上發生了一些小爭執，但那只是個小插曲。

從各方面來看，這真是無可挑剔的日常生活。

有多少人能過上這樣幸福的學院生活呢？如果她還抱怨，恐怕會被指責為貪得無厭。

然而，耶妮卡善良而真誠的性格帶來的問題從未得到解決。

這個根深蒂固的問題時不時會在她心中引發一種莫名的疼痛。

耶妮卡拿出自己準備好的文件，掃了一眼，又嘆了口氣。她猶豫了一會兒，最終把它扔進了附近的垃圾桶。

垃圾桶幾乎已經滿了，耶妮卡扔進去的文件滑到了頂部。這時，文件的內容才稍微映入她的眼帘。

——宿舍退宿申請表：德克斯館——

過了一會兒，她似乎又改變了主意，迅速從垃圾桶里把文件撿了回來。她再次左右張望，陷入了沉思……但最終還是沒有得出結論。

那些期待、羨慕、關懷、尊敬和尊重的目光，究竟有多大的重量壓在耶妮卡的肩上……甚至連她自己都不清楚。

她很難接受因為這些而感到壓力而選擇逃避的想法。畢竟，逃避通常是為了遠離痛苦或討厭的事物。

她怎麼可能把德克斯館的朋友們、職員們和室友們對她的期待和目光當作負擔呢？

然而，耶妮卡心裡明白。

從使用單人間的奧菲利斯館搬到多人間的德克斯館已經快一年了。

在這漫長的時間裡，耶妮卡真正能夠放鬆和休息的地方……最終只有埃德的營地。

* * *

[ 你不覺得感慨嗎……？去年的這個時候，我就站在你旁邊靜靜地看著你，而你卻毫無察覺。現在，你雖然以不完整的形式召喚了我，但至少我們可以交談了。成為契約關係後，感覺特別不一樣吧？]

「…….」

[當然，不一定非得是契約關係才能交談。埃德，你可能覺得這沒什麼大不了的，但你知道能夠駕馭高位精靈意味著什麼嗎……？這個學院的強者們……直斯？韋德？克萊維烏斯？如果我能發揮出全部力量，他們根本不算什麼~問題是我無法發揮出全部力量。]

「那麼……」

[ 當然，要達到那種境界，你需要很長的時間。不要和耶妮卡那樣的特例比較。不過，只要你繼續訓練，感應能力會越來越好，你就能更好地駕馭我了……嗯，是不是太遙遠了？不過沒關係，反正我們有的是時間。 ]

從北邊森林的營地出來，走向學院方向，會經過一個小湖。

湖中央的草島上，矗立著一棵巨大的「梅麗達的守護樹」……這裡是自然魔力最集中的地方，也是耶妮卡主要的精靈術訓練場所之一。

沒有比這裡更適合召喚不完全的梅麗達的地方了。

穿著白色連衣裙的白髮少女形象。雖然看起來像人類，但實際上並非如此。

我之所以不惜消耗大量魔力召喚梅麗達，是因為昨天和耶妮卡的對話。

她明確表示，提升與高位精靈的理解度會對我未來的戰鬥能力產生巨大影響。我也認同這一點。

理解度這個屬性，關鍵在於你與精靈的互動頻率、戰鬥中的使用次數、召喚次數以及對話的次數……但像這樣以人類形態召喚，幾乎耗盡了我的魔力，對我來說有些困難。

不過，既然召喚出來了，她自己喋喋不休地說話，倒是讓我輕鬆了不少。

耶妮卡曾評價她是世界上獨一無二的話癆。現在我才真正體會到她的評價是多麼準確。

<hr/>

[ 所以，你和耶妮卡怎麼樣了？那個互相尊重的敬語遊戲還在進行中嗎？ ]

「你不是都看到了嗎，還這麼問是什麼意思。」

[ 我可沒閑到跟著你們去學院……我只是好奇上課時有沒有發生什麼有趣的事情…… ]

梅麗達坐在守護樹粗壯的樹根上，晃著腿，帶著好奇的表情問道。

反正也沒必要撒謊……我決定把早上和耶妮卡發生的小爭執如實告訴她。

像往常一樣，早上在教授樓遇到耶妮卡時，我向她打招呼，結果她抱著橡木法杖嚇了一跳，連連後退。

顯然，昨天的餘波還在。總之，因為這件小事，我們之間又多了幾分尷尬。

[嗯~然後呢……？]

耶妮卡因為不知道該說什麼，臉紅著避開了視線，最後打了個招呼就消失了。

與此同時，敬語的使用還在繼續。

「一開始我並沒有太在意，但現在我開始覺得她內心可能有什麼壓力……或者『內在的壓力以某種奇怪的方式表現出來』。」

[ 哦，你在擔心耶妮卡嗎？ ]

「當然擔心。」

[ 哦~ ]

不知為什麼，梅麗達興奮地晃著腿，頻頻點頭。然後，她滿臉笑容地繼續問我。

[ 確實，她的狀態看起來有點不穩定。你得做點什麼。 ]

「所以我問你，能不能從這棵樹上折幾根樹枝……？」

[ 樹？突然要樹枝幹嘛？ ]

我和梅麗達並肩坐著的這棵守護樹，是一棵已經活了上千年的古樹。越是古老的樹，越能溫和地吸收和釋放魔力。

這棵梅麗達的守護樹，是製作魔工學物品的絕佳材料。如果再加上雷電魔法，它就能成為「雷擊千年木法杖」的完美材料。

比起耶妮卡手中那根老橡木法杖，它的性能和便利性都要高出許多。

我把這個事實簡要地告訴了梅麗達。

[ 嗯~原來如此……不過，嚴格來說，你不需要我的許可。 ]

「這棵樹不是你的樹嗎……？名字都叫梅麗達的守護樹……」

[ 嗯，這是個有點模糊的問題。只是很久以前的一個朋友把我的名字掛在這棵樹上而已。我自稱是這片森林的主人，但還不至於管到折一根樹枝這種小事。你隨意吧。]

「老朋友啊……也是，你活了這麼久，肯定有很多緣分。」

梅麗達晃著白色連衣裙，蹦蹦跳跳地繞著樹轉了一圈，然後回到了我身邊。

她抖了抖肩膀，擺出一副自以為嫵媚的姿態。

[我一開始就是以人類的形態出現的。我說過這並不容易吧？你覺得這是怎麼做到的？]

「怎麼做到的……你不是學會了相關的魔法嗎？」

[ 嗯……不太一樣。高位精靈可以變成各種形態，但最重要的是你能模仿得多熟悉、多舒服。完全變成世界上不存在的實體是非常困難的。大多數精靈都以動物的形態出現，因為它們在自然界中最常見。 ]

她微微一笑，提起裙擺，微微低頭行禮。這是帝國的禮儀。光看她的樣子，真的和人類沒什麼區別。

[ 所以，像這樣的人類形態也需要一個模仿的基準。既然我和你簽訂了契約，我就吸收了你的內心心理，變成了最接近你理想型的少女。 ]

「……什麼？」

[ 怎麼樣？人類的內心是無法欺騙的，看到這樣的我，你的心跳是不是加快了？你理想中的少女就站在你面前。 ]

我托著下巴，靜靜地看著梅麗達在我面前手舞足蹈。

「……我的理想型是這樣的……？確實……有點意外……」

[ 當然會意外。因為這是謊言。 ]

「…….」

你在開玩笑嗎？

帶著這樣的意味，我瞪大眼睛盯著梅麗達，而她則發出「哈哈」的輕快笑聲，抓著裙擺轉來轉去，像極了剛買了新衣服興奮不已的少女。

[ 我怎麼可能通過讀取內心心理來模仿呢？說實話，我是模仿了曾經見過的最特別、最獨特的人的模樣。像這樣……既不穿華麗的服飾，又總是洒脫隨性的樣子，給我留下了深刻的印象。把這棵樹命名為「梅麗達守護樹」的也是她。]

我再次看向梅麗達的人類形態。與那完全顯現時幾乎能摧毀一切的巨狼形態相比，差距太大了，我一時難以適應。

「是誰？」

當我問出這個問題時，梅麗達沒有立即回答。她只是意味深長地笑了笑。

[ 我活了很久。即使是西爾維尼亞學院的興衰，對我來說也不過是小事一樁。現在，阿肯島和西爾維尼亞學院似乎密不可分，但在我生活的遙遠過去，這裡甚至連學院都沒有，只是一座無人島。]

梅麗達解開了她像尾巴一樣紮起的頭髮。她的頭髮比我想像的還要濃密，散開後，她的形象完全改變了。

看到她的樣子，我不禁倒吸一口涼氣。那張臉讓我感到莫名的熟悉，彷彿在魔法史書中無數次看到過類似的面孔。

[ 怎麼樣？ ]

將魔力注入這棵樹，將其作為源泉。建立這所學校。請求梅麗達守護這片森林。這一切都是那麼久遠的人物所為。

原來，這一切都是這樣聯繫在一起的。

由於我對這些一無所知，即使在《西爾維尼亞的落第劍聖》劇本中，這些關係也從未被揭示。

眼前的少女是梅麗達不完全顯現的形態。她模仿的是誰，現在已經一目了然。

很久以前，那位比任何人都強大且睿智的魔法師和學者。

大賢者西爾維尼亞的模樣。

* * *

「聖皇陛下……要來訪？」

地點是學生會館旁邊的奧貝爾館，學生會辦公室。

塔雅·羅斯泰勒就任學生會長的職位並正式開始處理事務大約已經五天了。

雖然還處於適應期，但像這樣的重要職位，通常不會給你太多時間去慢慢適應。

從學生會書記那裡收到的報告文件中，最顯眼且被重點標註的部分。

位於特洛斯教團頂端的男人。統治聖都卡爾佩亞的聖皇埃爾登。

<hr/>

以及輔佐他的大主教貝爾迪奧。

即使是普通的主教級聖職人員來訪，也需要準備大量的禮儀程序，更何況是聖都的兩大巨頭同時來訪，光是聽到這個消息就讓人頭疼。

「訪問目的是……」

塔雅快速瀏覽了文件。她對速讀頗有自信。

訪問目的其實沒什麼特別的。傳播主神特洛斯的恩惠，迎接新受洗的信徒，順便在講壇上做些佈道。

不過，這顯然只是表面理由。

尊貴的特洛斯教團聖皇親自來到帝國偏遠的阿肯島……顯然是為了教團視若珍寶的聖女克拉麗絲。

聖女克拉麗絲是特洛斯教團的心臟。

即使是被神寵愛的僕從，也無法抵擋對知識的渴望，因此她在這個學期初來到了這片學習的土地——西爾維尼亞。

聖皇陛下……顯然是想親眼確認。

聖女克拉麗絲是否得到了應有的待遇，設施是否完善，是否存在危險。

如果有什麼不合格的地方，他甚至可能把克拉麗絲帶回去。

雖然令人遺憾，但最近西爾維尼亞事故頻發，出現幾個不合格的理由也並不奇怪。

「嗯……今天的工作結束後，我得安排時間去奧菲利斯館見見聖女殿下。書記伊蓮娜，沒問題吧？」

「好的。我會聯繫護衛人員。」

學生會書記又拿出了其他文件。還有很多事情要處理。

「啊，對了。聖都那邊還特別詢問了想單獨面談的學生。要轉交給學院處理嗎？還是您親自確認一下？」

「面談……？誰？聖皇陛下除了聖女殿下之外，還想見誰？」

「是的。是現二年級的學生，魔法部的阿黛爾·塞里斯……」

「阿黛爾……阿黛爾……我聽說過這位前輩。」

「是的……聽說她是個像風一樣的人物，不知道什麼時候會出現在哪裡。」

塔雅摸了摸下巴，沉思了一會兒。

「既然他們提出了請求，我們沒有理由拒絕。」

在這個時間點，聖都的人來到阿肯島。

感覺……並不愉快。

* * *

——啪嗒，啪嗒。嘩啦。

最終，我再次見到耶妮卡時，已經是深夜了。

僅僅因為一次敬語的交流，雙方都陷入了情感過載，幾乎花了一整天的時間來平復情緒。

深夜的營地。

我坐在篝火旁，一邊背誦基礎魔法公式，一邊用刀削蘋果。

現在，我的飲食已經可以兼顧營養均衡，攝入多種食物。

飯後吃水果也成了習慣，生活真的穩定了許多。

正當我悠閑地晃著腿，用刀輕輕敲打蘋果時……耶妮卡從草叢中出現了。

我停下削蘋果的手，看向耶妮卡。猶豫著該怎麼打招呼……最後決定先保持沉默。

「啊……你好……」

「…….」

「……您好？不對，你好？還是……您好……？」

她還在混亂中。到底是有多害羞啊。

不過，我也一時不知道該說什麼……只好慢慢點了點頭。說實話，我也有些尷尬。

就這樣，我們默默對視了幾秒，隨後耶妮卡害羞地慢慢走過來，小心翼翼地坐在篝火旁的石頭上。

我舉起削了一半的蘋果，問道。

「蘋果……要吃嗎？還是說……您要吃嗎？」

「好，好的……」

「…….」

水果刀繼續沿著原來的方向削著蘋果皮，發出輕微的沙沙聲。

「盤子……」

「這，這裡有……給……」

到底是用敬語，還是乾脆不用……能不能選一個？這樣至少能讓這令人窒息的氣氛稍微緩和一下。

耶妮卡還在急促地呼吸，頻頻轉頭深呼吸。她似乎以為我沒注意到。

耶妮卡接過盤子，拿起一個蘋果，像兔子一樣小口小口地啃了起來。

我也拿起一大塊蘋果咬了一口，然後開始削另一個蘋果。

又是一陣沉默。

「…….」

「…….」

突然，耶妮卡把臉埋在了膝蓋上。她的每一個動作都出乎意料。

「怎，怎麼了……耶妮卡……哪裡不舒服嗎……？」

「啊，不是。那個，有些話看著你的臉說不出口……所以，埃德，不，埃德先生。我有件事想問你，可能聽起來有點奇怪……所以，請聽完再回答我……好嗎……」

「…….」

耶妮卡把臉埋在膝蓋上，語速飛快地說著……我看了她一會兒，點了點頭。

然後我深吸一口氣，盡量裝作若無其事地繼續削蘋果。

看來，耶妮卡在氣氛變得奇怪後，自己跑出去想了很多。現在她來問我，顯然是她經過深思熟慮後做出的決定。

既然她這麼認真地思考過，那我應該盡量冷靜地傾聽。

<hr/>

無論她說什麼，我都不能露出驚訝的表情，不能慌張，即使真的感到困惑，也絕對不能表現出來。這點我還是能做到的。雖然身體還年輕，但我是個成年人。

我這樣想著，仔細聽著耶妮卡接下來的話。我下定決心，無論她說什麼，都要冷靜、簡潔地回應。

「如果……我從德克斯館搬出來……和埃德一起住在營地里……你覺得怎麼樣？」

——唰！！！

刀偏離了軌道，乾淨利落地削到了我的大拇指。

「…….」

鮮血從手指上流下，但我和耶妮卡都無暇顧及傷口。

黑暗的北邊森林營地。

發光的星空一如既往。

然而，一場微妙的變化悄然降臨。

我只能拿著蘋果和刀，獃獃地看著耶妮卡……

而耶妮卡則把通紅的臉深深埋在膝蓋里，試圖躲避我的視線。

就這樣，篝火的噼啪聲無情地打破了漫長的寂靜。
