# 第165章 風狼的憂鬱（1）

- 「很多精靈都覺得與人類交流很困難。因為人類的時間太短了，在精靈眼裡，他們總是顯得很急躁。」

- 「你說得好像自己不是人類一樣。在我看來，你也一樣。」

這是阿肯島學院建立之前的故事。

以高聳的奧倫山為基準，北邊的森林、東南邊的廣闊平原、西邊的海岸地帶，以及正南方的草原地帶。

在這些地方漫步時，尤其是在北端懸崖地帶延伸的闊葉林間穿行，或是沿著奧倫山的山腰攀登，或是沿著森林的溪流漫步時，常常能看到各種靈體精靈。

對於剛剛成為高位精靈的梅麗達來說，這溫馨的景象給她留下了深刻的印象。

因此，她常常在阿肯島停留，不知不覺間，她已經完全定居在這裡，不再離開。

儘管阿肯島位於大陸西南角的盡頭，但這裡並非沒有原住民。

從扎赫爾邊境伯爵領地逃出、尋找新家園的移民，或是為了阿肯島海岸洞穴中的魔力石等天然資源而來的探險家們，時常會來到這裡。

也有一些有故事的定居者來到這裡，形成了大約六七戶人家的小村莊。

人類這個種族，有時似乎與自然和諧相處，但必要時也會威脅自然……他們是難以捉摸的存在。對於不知不覺間愛上了這片北邊森林的梅麗達來說，他們也是難以應對的對象。

雖然無法一一細說，但這裡既有衝突，也有和諧的時刻。有時大家會合力應對島上的災難，有時梅麗達也會對過度狩獵和伐木的人類感到憤怒。

- 「我要建一所學校。」

在這些人中，最令她印象深刻的怪人，就是那個少女。

她將閃亮的銀髮紮起，穿著魔法師們常穿的皮製束腰外衣，披著金色的半長袍，總是充滿活力。她手上戴著幾枚樸素的戒指，脖子上掛著龍牙形狀的項鏈，衣擺上還裝飾著寶石，每一件都充滿了魔力。

這位在遙遠的北方被稱為「大賢者」的少女，因在皇室研究設施中使用禁忌魔法而被驅逐……傳聞她是個怪人中的怪人。

儘管她過去的成就令人矚目到皇室無法追究她的罪責，但對於這位充滿好奇心和學術熱情的少女來說，被流放到這個一無所有的島上已經是足夠讓她感到憤怒的事了。

然而，落後的環境並不能阻擋這位著名的魔法研究者——西爾維尼亞·羅貝斯特的熱情。

她剛來到阿肯島時，砍伐北邊森林的樹木製作魔法杖，研究物質轉換魔法，常常與梅麗達發生衝突。大部分時間都在爭吵和打鬥中度過。

她突然在河岸建造堤壩，試圖將水的下落能量轉化為魔力，或是捕捉精靈研究封印法陣，甚至試圖在巨大的石碑上刻下禁忌魔法之一的「交換魔法」。

她那難以捉摸的行事風格讓梅麗達頭疼不已，但西爾維尼亞始終微笑著，研究各種類型的魔力。

在這個連一個研究用試管都沒有的地方，西爾維尼亞寫下了大量足以載入史冊的研究著作。

梅麗達再怎麼勸阻，也無法熄滅她的學術熱情。

儘管她總是做些難以理解的事情，梅麗達已經習慣了，但這次她居然說要建一所學校。

- 「嗯，規模很小，與其說是學校，不如說是個學堂。」

- 「你又開始說些莫名其妙的話了。」

西爾維尼亞總是喜歡惹事，自從她來到阿肯島後，沒有一天是平靜的。

對於常常在北邊森林的湖邊或奧倫山腰悠閑吹風的梅麗達來說，這段時間簡直是折磨。

- 「我打算在東邊的平地上建一座小木屋，教魔法和基礎戰鬥技巧。如果可以的話，順便也教鍊金術。」

- 「你真是閑得發慌啊。」

- 「不，我不是常說嗎？和精靈不同，人類的時間有限，所以總是忙忙碌碌的。」

西爾維尼亞雙手叉腰，露出狡猾的笑容，表情已經讓人覺得不安了。

- 「從皇室或無法地帶流落到阿肯島的孩子們有幾個。他們都是了不起的孩子，皇室大概是覺得讓他們別惹事，所以讓我來負責照顧他們。每個孩子都有自己的故事。」

- 「也是，你被流放到阿肯島後也惹了不少麻煩。皇室大概是給你找點事做吧。」

- 「嗯，雖然覺得他們是在讓我帶孩子，心情有點不爽……但意外的是，這些孩子都很善良和勤奮。而且他們的出身也都不簡單，果然交給我這個『大賢者』是有原因的~」

西爾維尼亞一邊說著，一邊列舉了即將來到阿肯島的孩子們。

初代劍聖魯登的後裔——泰斯林·麥克羅爾，受到星辰祝福的少年魔法師——格洛克特·埃爾德貝恩，以及在成人禮前就製造了「致命藥劑」的女鍊金術士——菲洛娜·布魯姆里弗。

這三位在戰鬥、魔法和鍊金術上各有所長的孩子，作為大賢者西爾維尼亞的首批弟子，未來將在大陸上叱吒風雲。

- 「總之，我打算在東邊的平地上建個學堂，梅麗達你肯定會鬧騰，所以提前來跟你打個招呼~。北邊森林附近我絕對不會碰，繼續保持自然狀態，你也裝作不知道吧~」

阿肯島的風景更像是自然的土地，而非學術的領地。

無論人類世界如何變化，梅麗達只是靜靜地坐在奧倫山頂，看著樹木搖曳。

在她面前，那位大賢者心情似乎特別好，總是哼著小曲。以前覺得她煩人，但最近似乎已經習慣了，覺得無所謂了。

適應真是可怕。

就像滴水穿石一樣，不知不覺間，這個少女已經融入了她的生活……變得理所當然。

梅麗達通過這樣的經歷，意識到與人類親近或許並不是什麼大不了的事。

與人類交流並不壞，只是比想像中更麻煩。

帶著這種無聊的感慨，狼靜靜地俯瞰著阿肯島的風景。

精靈的時間與人類截然不同，流逝得飛快。

太陽從東邊的海面升起，又在西邊的海面緩緩落下。隨著這樣的循環，島上的風景也在不斷變化。

櫻花盛開，草木繁茂，落葉飄零，最後積雪覆蓋。阿肯島的每一刻都美麗，但並非一成不變。

東邊那座小小的學堂逐漸擴大規模。進出的學生越來越多，不知不覺間，宏偉的建築拔地而起，尖塔直指天空。這座起初只是「學堂」的地方，如今已被稱為「西爾維尼亞學院」，紀念著那位已不在人世的少女的成就。

就這樣，東邊的學堂隨著歲月的流逝，成為了「教授樓」。

西邊那幾戶人家的小村莊，隨著學堂規模的擴大，人口也逐漸增加……不知不覺間，形成了自己的商業區，許多人開始來到這裡尋找機會。

於是，西邊的村莊被命名為「生活區」，學院管理的宿舍也開始建立。

時間流逝，阿肯島的風景也發生了巨大的變化。

曾經只有野生動物出沒的阿肯島，如今已成為遙遠的過去。

偶爾出現的建築，或是學院里來來往往的學生，讓人真切地感受到時代的變遷。

各種武器也得到了發展，戰鬥技巧和戰術研究也取得了長足進步，戰鬥部的課程分化為數十個分支。如今，學院戰鬥部不僅培養戰士，還培養出了著名的將軍和謀略家。

魔法領域也是如此。曾經掌握一個中位魔法就能被稱為精英魔法師的時代已經過去，最近甚至出現了在學院級別就能使用最高位魔法的魔法師。雖然她的實力過於異常。

煉金領域則朝著完全不同的方向發展，從藥劑調配和礦物學擴展到魔工學、草藥學、工藝學和魔法生物學，幾乎涵蓋了所有與魔力相關的領域。魔法無法實現的一切，似乎都被煉金領域吞噬了。

時代的變遷和學術的進步是如此迅速而輝煌。

[時代就這樣將我拋在身後，繼續前進。]

在奧倫山腰一棵特別高大的樹上，化為人形的梅麗達坐在那裡。

<hr/>

看著西爾維尼亞學院的風景，梅麗達低聲吟誦。

[ 一如既往。 ]

*

[ 我要離家出走了。 ]

聽多了突如其來的話，也會產生免疫力。

「...這又是什麼話？」

[ 最近大家都太把我當回事了，所以我打算消失一段時間。算是某種自我價值的維護吧。呵呵。 ]

「...」

營地的火焰在白天的溫暖陽光下燃燒。

雖然是夏末，但天氣已經相當涼爽。不過，要說秋天已經到來還為時尚早，正是那種穿得薄了容易因為溫差而感冒的季節。

對我來說，感冒已經不是問題了。因為我的身體已經糟糕透了。

當我坐在篝火旁發獃時，貝爾·梅亞正在工作台那邊切洋蔥。

只有菜刀敲擊砧板的聲音傳來，氣氛十分寧靜。

「...您叫我了嗎？」

「不，是梅麗達……就是和我契約的精靈，說了些無聊的話。」

「原來如此。」

我的魔力已經耗盡，無法顯現出梅麗達的人形。

所以，沒有精靈感應能力的貝爾看不到梅麗達。

貝爾簡短地回答後，開始整理籃子里的一些食材。

因為是假期，奧菲利斯館的工作相對輕鬆。

她有空時會來照顧身體不好的我，順便處理一些營地維護的雜務，對此我非常感激。

多虧了她，我可以在假期結束前好好休息。等身體好一些，我打算好好報答她。

[ 無聊的話？我可是經歷了很多苦惱呢~。 ]

梅麗達從篝火對面的樹榦上輕巧地跳下來，純白色的裙子隨風飄動，赤腳輕輕一蹬，身體自然地浮起，飄到了我旁邊的樹榦上。

她輕巧地坐下，將雙腿伸向篝火，露出狡黠的微笑。

[ 怎麼說呢，突然就想離開一下。如果我突然消失，北森林沒人管理，那些在這裡安家的靈體精靈和下位精靈們肯定會驚慌失措吧？ ]

「你性格真差。」

[ 我只是定期進行自我價值管理而已。善意持續太久，別人就會覺得是理所當然~。然後就會變成冤大頭。我得定期提醒他們，我有多辛苦，沒有我會有多麻煩。嗯。 ]

梅麗達咯咯笑著，將連衣裙的下擺拉直。

[ 反正你剩下的假期也只是休息吧？我打算現形一下，去學院里轉轉。這幾天我打算把北森林丟在一邊~。 ]

「簡單來說就是想休息一下。好吧，這沒什麼問題...但露西現在在皇室，你打算用誰的魔力來現形？」

平時梅麗達會從露西和我這裡平分魔力，但現在在阿肯島上，沒有足夠的地方供她獲取魔力。

一旦平時積累的魔力耗盡，她就無法維持現形狀態。上次在克雷平戰役中，她自己積累的魔力也幾乎用光了，現在可以說是身無分文。

[ 你不是有法杖嗎？雖然魔力不足時我無法發動精靈術士，但僅僅維持人類形態的現形，法杖的自身能力就足夠了。反正我也不需要戰鬥能力，所以問題不大。 ]

「...所以你才突然跟我說這些莫名其妙的話。是想借我的法杖用用吧。」

[ 我可是在離家出走前也會好好徵求同意的乖孩子~。 ]

世界上哪有徵求同意後再離家出走的瘋子。

不過，考慮到她是受契約約束的精靈，這種特殊身份也是無可奈何。

[ 我打算去學院里轉轉，看看風景，所以如果沒什麼急事的話，就當我有點流浪癖，別管我了。 ]

「好吧。你平時也辛苦了，該休息的時候就休息吧。」

我因為身體虛弱，拜託貝爾幫我把法杖拿來。

貝爾在篝火旁煮著鍋里的水，輕輕點了點頭，然後走進了小屋。

[ 耶妮卡和露西大概要等到假期結束才會回來吧？在那之前，你好好養傷吧。埃德，你也傷得不輕。 ]

「把契約者丟在營地里，自己跑去流浪的精靈有資格說這種話嗎？」

[ 反正那個女僕一直在照顧你。她心裡肯定有不少壞心思，只是我在旁邊，她才不敢表現出來吧？ ]

梅麗達狡黠地笑著，我則瞪大眼睛，靜靜地看著她的身影。

[ 玩笑開不下去了。 ]

「我現在不休息，就沒機會休息了。」

[ 是啊。假期之前，家族殘餘勢力的消息也會傳來，學期開始後又要忙著應付課程了。哎呀哎呀~，真是忙得夠嗆呢~。 ]

梅麗達調皮地抬起赤腳，輕輕踢了幾塊附近的石頭。幾塊小石頭在草叢中滾了幾圈，最終停在了篝火旁邊。

就這樣發了一會兒呆，貝爾從小屋裡拿出了我的法杖。

「給您。您是要使用魔力嗎？最好不要太過勉強自己。」

「不，反正我也沒有魔力可用。只是依靠法杖的能力而已 。」

我握住法杖，稍微集中精神，法杖的力量開始慢慢顯現。

雖然完全無法發揮戰鬥能力，但梅麗達還是成功地以少女的形態顯現了出來。

她真的只是顯現出來而已，沒有任何魔力波動。只是一個漂亮的白髮少女。

這樣的少女在木樁上顯現出來，貝爾好奇地看了一會兒。梅麗達對貝爾來說是熟面孔，所以她自然地露出迷人的微笑，從木樁上跳了下來。

[ 啊哈，我顯現的時候踩在草地上的感覺真不錯。 ]

雖然她的腳看起來纖細得讓人擔心會不會受傷，但梅麗達毫不在意地提起裙擺，輕盈地跳了幾下。

[ 那在本家的消息傳來之前，你就好好休息吧~。我也要出去逛逛了~。 ]

梅麗達說完，朝我揮了揮手，蹦蹦跳跳地離開了。

光著腳那樣走不會痛嗎？

<hr/>

雖然有些疑惑，但我還是隨她去了。

我用火鉗戳了戳篝火，深深地嘆了口氣，放鬆了身體。

味覺消失了。所有的食物都像嚼沙子一樣。一隻耳朵聽不清，左腿也使不上力。

間歇性的低燒和高燒交替發作，時不時還會感到噁心或發冷。

魔力已經一滴不剩了。

我甚至沒有完全召喚出最高位精靈。只是召喚了它的頭部，發射了一次火焰而已。

僅僅那短暫的顯現就讓我的身體垮成這樣，而耶妮卡長時間召喚最高位水精靈，她的感應能力到底有多強？

當然，耶妮卡在召喚弗里德後也躺了很久，需要照顧，但即便如此，她的感應能力也非同尋常。

「...」

想到耶妮卡，她登上前往普蘭的列車時那擔憂的表情也浮現在腦海中。洛特爾和克拉麗絲也一樣。

然而，我不能讓他們為了照顧我而推遲寶貴的假期或工作/公務安排。

最好的回報就是在假期結束時，以恢複良好的狀態迎接他們。

最讓我擔心的是塔雅。

她披著象徵家主的鷹徽斗篷，推著我的背讓我回阿肯島。

閉上眼睛，紮起頭髮、望著倒塌的羅斯泰勒宅邸的塔雅的背影浮現在腦海中。

雖然擔心，但在精神恍惚和身體虛弱的狀態下，我什麼都做不了。幫忙反而可能成為負擔。

儘快恢複是最重要的。

深知這一點，我盡量放鬆身體……把毫無味道的食物硬塞進嘴裡。

*

北方草原地帶的軍團長，馬格努斯·凱拉莫爾到達羅斯泰勒宅邸時，一切已經結束了。

他原本計畫在社交會議結束時來訪，但因軍部會議突然召開而未能及時趕到。

他看著完全倒塌的羅斯泰勒宅邸和臨時搭建的帳篷，不由得瞪大了眼睛。

「馬格努斯軍團長，招待不周，實在抱歉。」

在最大的帳篷中央，坐著一位正在處理事務的少女。

她是西爾維尼亞學院的學生會長，也是現任羅斯泰勒家族的家主——塔雅·羅斯泰勒。

她稚嫩的臉上還帶著一絲疲憊，但塔雅依然熟練地與比她年長許多的軍團長交談。

「如您所見，羅斯泰勒宅邸現在的情況……」

「發生了什麼事？」

馬格努斯捋了捋濃密的鬍鬚，恭敬地問道。他立刻意識到她的身份已經提升。

塔雅苦笑著低下頭。

塔雅周圍還留著幾位討論宅邸情況的貴族。雖然大部分已經返回各自的領地，但一些核心貴族仍在現場處理事務。

其中之一是布魯姆里弗家族的家主——西尼爾·布魯姆里弗。

這位年邁的女士向馬格努斯微微點頭。

馬格努斯也點頭回禮，然後坐在塔雅面前。

西尼爾和馬格努斯都是將孩子送到西爾維尼亞學院的父母。

馬格努斯的兒子，韋德·凱拉莫爾，是一年級的首席。而西尼爾的雙胞胎女兒中的次女，特蕾西亞娜·布魯姆里弗，是四年級魔法部的首席。

作為優等生的父母，他們對西爾維尼亞學院的學生會長塔雅·羅斯泰勒也有所了解。

不過，他們從未真正見過她，只能在心裡猜測她是怎樣的少女。

這位負責管理西爾維尼亞學院學生的學生會長，同時還繼承了瀕臨崩潰的家族的家主之位。

肩負如此重任的少女，身材卻比想像中瘦小，兩人不由得用擔憂的目光看著她。

然而，塔雅毫不在意地開口了。

「長話短說，我現在處境艱難。沒時間試探了。」

對於尚未了解羅斯泰勒宅邸情況的馬格努斯來說，這話聽起來有些莫名其妙。

但塔雅直截了當地說道。

「我要扳倒塞拉哈皇女。」

在塞拉哈、佩爾西卡和佩妮亞的三方皇權爭奪中，塔雅必須找到生存的方向。

「您願意合作嗎？」

馬格努斯，這位繼承了戰士血脈、統領北方草原的大將軍。

西尼爾，掌管通往女巫之家的布魯姆里弗家族的家主。

要拉攏這兩人，必須有名分或權力。

然而，塔雅現在是一個瀕臨崩潰的家族的家主。沒有人會追隨一個陷入絕境的人。

她並非不知道這一點，但她依然堅定地說道。彷彿這是唯一的答案。
